◆ I2C_MODE_ET

enum I2C_MODE_ET

This Enum , defines the different modes of I2C that you can set in the PIC Microcontroller.

Enumerator
I2C_MODE_SLAVE_ADDR10_INTEN 

I2C Slave with 10 bit Address and Interrupts Enabled.

I2C_MODE_SLAVE_ADDR7_INTEN 

I2C Slave with 7 bit Address and Interrupts Enabled.

I2C_MODE_MASTER_FIRMWARE_CONTROLLED 

I2C Master , this Mode requires the user to define his own I2C Functions , the hardware only helps by detecting bus activity.

I2C_MODE_MASTER_HARDWARE_CONTROLLED 

I2C Master , Hardware Controls the actual bus transfers , and user generates conditions like start,stop,etc .

I2C_MODE_SLAVE_ADDR10 

Slave Mode with 10 bit address and no interrupts enabled.

I2C_MODE_SLAVE_ADDR7 

Slave Mode with 7 bit address and no interrupts enabled.

void I2C_Halt ( void  )

: Blocks until the i2c bus is idle again .

This Function is used to Block(halt) execution until the i2c bus is idle . Functions using the I2C Module for Transmit/Receive Should Call this Function Before taking an action on the bus to ensure that the bus is idle beforehand. This function is blocking.
